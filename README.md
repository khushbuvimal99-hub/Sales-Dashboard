# 📊 Sales Analytics Dashboard (Excel Project)

## 🔍 Project Overview

This project presents a dynamic Sales Analytics Dashboard built using Microsoft Excel. It provides insights into sales performance, product trends, and regional contributions through interactive visualizations.

## 🎯 Objective

To analyze sales data and help businesses make data-driven decisions by tracking revenue, orders, and product performance.

## 🛠 Tools & Techniques Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers (Category, Region, Month)
* Data Cleaning & Transformation

## 📊 Key KPIs

* 💰 Total Sales: ₹52,053,100
* 📦 Total Orders: 1001
* 📈 Total Quantity Sold: 2975

## 📈 Dashboard Features

### 🔹 Monthly Sales Trend

* Displays sales performance across months
* Helps identify peak and low-performing periods

### 🔹 Sales by Region

* Compares sales across regions (North, South, East, West)
* Identifies top-performing regions

### 🔹 Top Selling Products

* Highlights best-selling products
* Supports product-level decision making

### 🔹 Category Contribution

* Shows percentage contribution of product categories
* Helps in understanding revenue distribution

## 🎛 Interactive Filters

* Category (Accessories, Electronics)
* Region (North, South, East, West)
* Month (Jan–Dec)

## 📌 Key Insights

* Sales peak observed during mid-year months
* West region contributes highest revenue
* Electronics category dominates overall sales
* Certain products consistently outperform others
